# 🇧🇷 Português

## 👨🏽‍💻 Sobre mim

Sou **Erick Carrasco Oliveira**, também conhecido como **CARRASCO**.

Sou **CEO e fundador da NOOK**, uma edtech dedicada ao desenvolvimento de **soluções educativas inclusivas** para melhorar o ambiente de aprendizagem de professores e estudantes.

Minha trajetória conecta **tecnologia, educação e inovação social**.  
Iniciei minha jornada empreendedora ainda no ensino médio, quando fundei minha primeira startup aos **17 anos**, motivado pela ideia de usar tecnologia para resolver problemas reais na educação.

Sou formado em **Análise e Desenvolvimento de Sistemas pelo Senac** e atuo como **técnico em desenvolvimento de software**, desenvolvendo aplicações web e soluções digitais.

Acredito que:

> Educação não deve apenas formar alunos —  
> deve formar pensamento, autonomia e protagonismo.

## 🚀 NOOK
